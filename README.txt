ШАГ 1: СОЗДАТЬ БОТА
  1. Найдите @BotFather в Telegram
  2. Отправьте /newbot
  3. Имя: Inspection Bot
  4. Username: company_inspection_bot
  5. Скопируйте токен

ШАГ 2: ЗАГРУЗИТЬ ФАЙЛЫ НА ХОСТИНГ
  1. Откройте app.netlify.com/drop
  2. Перетащите папку с 4 файлами
  3. Скопируйте ссылку (https://...netlify.app)

ШАГ 3: НАСТРОИТЬ WEB APP
  1. В @BotFather: /mybots → выберите бота
  2. Bot Settings → Menu Button → Configure
  3. Отправьте ссылку на веб-приложение
  4. Название кнопки: 📋 Начать проверку

ШАГ 4: ПОЛУЧИТЬ CHAT_ID
  1. Добавьте бота в чат с менеджером
  2. Отправьте любое сообщение
  3. Откройте: https://api.telegram.org/bot<ТОКЕН>/getUpdates
  4. Найдите "chat":{"id": -1001234567890}

ШАГ 5: ВСТАВИТЬ ДАННЫЕ В app.js
  Строка ~370: const botToken = 'ВАШ_ТОКЕН'
  Строка ~371: const chatId = 'ВАШ_CHAT_ID'
